## Installation

Node Version: v16.14.0

```bash
$ yarn install
```

```bash
$ npm i -g @nestjs/cli
```

```bash
$ docker-compose up -d
```

## Clone file

```bash
$ cp .env.example .env
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
